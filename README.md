# socket-io-server

FlutterとSocket.IOを使用したリアルタイム通信のデモアプリのサーバー

## セットアップ

```bash
$ git clone https://github.com/takashi0602/socket-io-server.git

$ cd socket-io-server

$ npm install

$ npm run dev
```
